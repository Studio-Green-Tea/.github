  # 這裡是 **綠茶字幕組**

  綠茶漢化組是由一群熱愛ACG文化的愛好者組成的非營利性組織，成立於2017年10月4日。下屬綠茶字幕組（綠茶漢化組動畫字幕分組）成立於2025年7月22日，旨在為觀眾提供準確、一流的字幕。

  本組長期招收時軸、特效、翻譯、美工、校對、分流、壓制等崗位，招募測試QQ群：[綠茶·雲澗·雪嵐·青山🍵](https://qm.qq.com/q/cRSmEuOi4g)
