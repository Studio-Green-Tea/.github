# 贡献指南

感谢您帮助绿茶字幕组改进字幕和仓库内容。

## 选择提交方式

- 如果您只是发现了错字、错译、错轴、文件缺失或下载问题，请优先提交[问题反馈](https://github.com/Studio-Green-Tea/Studio-GreenTea-Subtitle-Storage/issues/new?template=problem-feedback.yml)。
- 如果您已经准备好明确的修正内容，可以提交 Pull Request。

## Pull Request 要求

1. 每个 Pull Request 尽量只处理一类问题或一部作品。
2. 请写明涉及的作品、集数、文件路径和时间点，并说明修改前后的内容。
3. 翻译修正请尽量附上可靠的参考来源或理由。
4. 请勿提交视频、封装成品、未经授权的字体、图片或其他第三方文件。
5. 提交前请检查字幕编码、文件名和目录结构，避免加入无关文件。

## 贡献授权

提交 Pull Request 即表示您确认：

- 您创作了相关贡献，或已经取得提交和授权该内容所需的权利；
- 您授权绿茶字幕组及本组织的仓库维护者审阅、编辑、合并、复制并发布该贡献；
- 贡献被合并后，将作为仓库内容按照仓库标明的许可条件发布，除非双方另有书面约定；
- Pull Request 只是向官方仓库提出贡献，不代表任何人获得在仓库之外发布字幕修改版的许可。

如果您不同意以上授权，或不能确认相关权利，请仅提交 Issue，不要提交 Pull Request。

## 许可范围

仓库所标示的 CC BY-NC-ND 4.0 许可，仅适用于绿茶字幕组或贡献者有权许可的翻译、时间轴、字幕样式和其他原创内容。动画原作、原始台词、商标、字体、图片及其他第三方材料仍归各自权利人所有，不因存放在本组织仓库中而获得重新授权。

“禁止修改”是指未经单独授权，不得在仓库之外向公众发布字幕修改版；这不妨碍贡献者通过 Issue 或 Pull Request 向官方仓库提出修正建议。
